# 🗺️ VeriMap - 實時網路商家信譽與評論可信度分析系統

VeriMap 是一款由 AI 驅動的 **Google 地圖商家評論打假與防抹黑分析系統**。它結合了最先進的 **Gemini 2.5 聯網搜尋技術 (Google Search Grounding)** 與 **40%/30%/30% 分層抽樣技術**，能實時過濾水軍洗板與同行惡意負評，還原商家最真實的顧客體驗痛點，並重新計算出最客觀的「AI 綜合實際星等」。

本專案包含：
* **`verimap/backend`**：基於 FastAPI 與 Google GenAI 官方最新 SDK 構建的高性能結構化分析後端。
* **`verimap/frontend/app`**：基於 Flutter 構建，具備極致美感（玻璃摩砂質感、Bento 佈局、動態統計與多帳號聯動）的跨平台前端 App。

## 🎯 核心亮點與特色

### 1. 🔍 即時雙階段聯網搜尋與分析 (Two-Step Grounding Workflow)
* **第一階段（聯網採樣）**：當輸入 Google 地圖網址時，後端會自動解碼並提取乾淨的商家名稱，自動發送 4-5 組不同的搜尋組合（包含差評、普通、好評、Dcard/PTT 爭議等），抓取並大量閱讀 **50 到 100 則** 真實評論大樣本。
* **第二階段（結構化分析）**：將採樣內容送入 Gemini 2.5 模型進行 NLP 分析，嚴格匹配 Pydantic 定義的 JSON 結構輸出，保證 VeriScore 計算百分之百精準。

### 2. ⚖️ 40/30/30 分層抽樣過濾與防惡意抹黑
* **40% 低星投訴分析**：自動過濾無實質內容的同行抹黑、水軍差評與無理情緒宣洩，精準還原核心服務與環境衛生真實痛點。
* **30% 中星客觀體驗**：提取中立顧客好壞並陳的真實就餐/就醫感受。
* **30% 高星好評分析**：評估正面評價中，有多少屬於自然的真實讚美，多少屬於店家「打卡送小菜/好評送折扣」等行銷利益引誘的灌水好評。
* **同行差評洗板檢測**：估計低星差評中屬於「惡意/無理/抹黑」的佔比，並警示商家是否正遭受惡意攻擊。

### 3. ⭐ AI 綜合實際星等 (AI star rating)
* 在排除行銷洗分、無字水軍好評，並過濾掉惡意差評雜訊後，由 AI 重新加權估算出該商家的 **「真實滿意度星等 (1.0 - 5.0 星)」**，讓消費者不再被洗出來的 4.9 高分欺騙。

### 4. 🔄 全動態前端帳號連接與收藏聯動
* **實時統計**：個人檔案頁面的「已分析商家數」與「我的收藏」與後端及歷史紀錄 100% 動態連線，任何操作皆會立刻重繪 App 介面。
* **已連接帳號設定**：支持動態連接 Apple ID 與 LINE 帳號，並帶有平滑的開關動畫與 SnackBar 交互提示。
* **自訂分析偏好**：可彈出對話框調整預設抓取評論數 (50, 80, 100 則) 及 AI 痛點焦點 (全部, 服務態度, 衛生清潔)。

## 🚀 快速啟動指南

### 第一步：啟動 FastAPI 後端

1. 打開終端機，進入後端目錄：
   ```bash
   cd verimap/backend
   ```
2. 安裝 Python 依賴套件（建議使用虛擬環境）：
   ```bash
   pip install -r requirements.txt
   ```
3. 在 `verimap/backend` 資料夾下建立 `.env` 檔案，並填入您的 Gemini API Key：
   ```env
   GEMINI_API_KEY=your_actual_gemini_api_key_here
   ```
4. 啟動後端本地伺服器（預設為 8000 埠）：
   ```bash
   uvicorn main:app --reload --port 8000
   ```
   *啟動成功後，您可以瀏覽 [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs) 查看並測試 Swagger API 文件！*

---

### 第二步：啟動 Flutter 前端 App

1. 打開另一個終端機視窗，進入前端 App 目錄：
   ```bash
   cd verimap/frontend/app
   ```
2. 獲取 Flutter 依賴包：
   ```bash
   flutter pub get
   ```
3. 以 Chrome 瀏覽器調試模式運行應用程式（或使用您習慣的模擬器/桌面端）：
   ```bash
   flutter run -d chrome
   ```
   *前端會自動連線至 `http://127.0.0.1:8000/analyze` 進行實時打假分析！*

## 🧪 功能驗證與完整測試流程

啟動前端 App 後，您可以依照以下步驟驗證系統的動態聯動：

1. **📊 查看初始 Profile 數據**：
   * 點擊 App 底部右下角的 **「個人檔案」** 分頁。
   * 此時可以看到初始動態統計顯示為：**「已分析商家數：4」**、**「我的收藏：2」**（對應預設的「錦錨餐廳」與「極品燒肉」）。

2. **👤 編輯個人資料與帳號狀態**：
   * 在個人檔案頁面點擊 **「編輯個人資料」**，輸入新名字（例如：張大同）與新信箱，點擊儲存。您會看到個人卡片上的名字與信箱**瞬間完成更新**！
   * 滑動到下方的 **「已連接帳號設定」**。點擊切換 **Apple ID** 與 **LINE** 的開關。系統會展現平滑的開關動畫，並彈出 SnackBar 回報已連接/解除連接的狀態，且該狀態會保存在 App 的靜態快取中。

3. **⚙️ 調整分析偏好設定**：
   * 在下方選單點擊 **「分析偏好設定」**。
   * 您可以自由切換預設抓取的評論數量（50, 80, 100 則）以及痛點偏好焦點（全部, 服務態度, 衛生清潔），點擊儲存後設定將即時套用。

4. **📂 歷史紀錄收藏聯動**：
   * 切換到底部導航的 **「歷史記錄」** 分頁。
   * 點擊未收藏的商家（如「老街豆花」）打開底部的詳細分析報告。
   * 點擊左下角新增的 **黃色收藏 (Bookmark) 圖示按鈕**，將其標記為收藏。
   * 關閉詳情彈窗，回到 **「個人檔案」** 分頁，您會看到「我的收藏」從 `2` **自動遞增為 `3`**！

5. **🔍 首頁實時聯網打假與收藏**：
   * 切換到 **「首頁」** 分頁，直接點擊下方快捷測試 preset 裡的 **「中壢長榮醫院」**。
   * 系統會彈出加載對話框，展示「VeriScore 正在深度分析中...正在啟動聯網搜尋...」。
   * 後端將自動在 Google 上執行多次多維度搜尋，抓取 80 則真實評論進行打假，約 10-15 秒後呈現精緻的報告：包括 **AI 綜合實際星等 (如 3.4/5.0 星)**、可信度儀表板針尖指向、**40%/30%/30% 的分層抽樣打假文字分析**、以及**環境衛生/服務態度痛點提取**。
   * 點擊報告頁面右上角新加的 **收藏圖示**。
   * 回到 **「個人檔案」** 分頁，您將見證：
     - **已分析商家數** 自動從 `4` **更新為 `5`**！
     - **我的收藏** 自動從 `3` **更新為 `4`**！

6. **🧹 一鍵清除快取還原**：
   * 在個人檔案選單底部點擊 **「重設帳號狀態與快取」** 並確認，即可隨時將所有收藏狀態、個人資料與偏好重置為最初始的 Demo 預設值，方便重複演示！
